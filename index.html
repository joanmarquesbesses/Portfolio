Runic2D

Runic2D is a custom-built 2D game engine as part of my Final Degree Project (TFG), developed from scratch in C++ with OpenGL, designed with a strong focus on performance and optimization. It features a multithreaded architecture for efficient resource loading and execution, along with advanced rendering optimizations such as batch rendering to maximize frame rates. The engine includes a custom event system, a physics module for real-time simulations, and support for flexible shader and texture management. Built with modularity in mind, Runic2D is designed to be easily extensible and adaptable across different platforms and rendering backends, making it a solid foundation for building fast and scalable 2D games.
